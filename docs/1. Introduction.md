## Introduction

Greygoose performs the following steps in order.

- Read files from source folder
- Parse frontmatter
- ?? Substitute variables
- Compile file
- Processing
- Write files



## Read files from source folder

source folder defaults to `src`, configurable
ignore if filename begins with underscore, configurable, e.g. `config.ignoreIfFilename("_", ...)`
ignore if custom ignore rule, e.g. `config.ignoreIfPath("/path/to/file", ...)`

done using JS



## Parse frontmatter

parsed as YAML, configurable, e.g. `config.parseFrontmatter("YAML")`

done using gray-matter



## ?? Substitute variables

configurable, `config.variableTransform("data", data => {... return ...})`

done using JS



## Compile file

markdown to HTML, configurable, e.g. `config.parse({from: [".md", ".js"], to: ".html"}, {...})`

done using markdown-it



## Processing

configurable, `config.htmlTransform(...)`

done using JS

done using JS
